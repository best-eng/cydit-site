# Cydit site

Чистый статический лендинг Cydit — без зависимостей и сборщика. Двуязычный (RU/EN), тёмная тема.

## Запуск

Откройте `index.html` в браузере или разместите папку `cydit-site` на любом статическом хостинге (GitHub Pages, Netlify, Cloudflare Pages и т.п.).

## Структура

- `index.html` — разметка секций.
- `app.js` — контент (RU/EN) и рендер: hero, метрики, фичи, «как это работает», экраны, сравнение, роадмап, FAQ, CTA.
- `styles.css` — стили и адаптив.
- `assets/` — логотип и скриншоты.

## Язык

Переключатель RU/EN в шапке; выбор сохраняется в `localStorage`.

## Лист ожидания

Форма раннего доступа в секции «Early access» работает на фронтенде и показывает подтверждение. Чтобы реально собирать заявки, подключите обработчик `#waitlistForm` в `app.js` к своему backend или форм-сервису (Formspree, Google Forms, собственный API).
