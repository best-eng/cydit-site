# Cydit site

Чистый статический сайт Cydit без зависимостей и сборщика.

## Запуск

Откройте `index.html` в браузере или разместите папку `cydit-site` на любом статическом хостинге.

## Админ-доступ

Админ-панель скрыта и появляется только после входа под админ-аккаунтом.

Контент сохраняется локально в `localStorage`. Для production CMS можно заменить слой `localStorage` в `app.js` на API и перенести проверку админ-доступа на сервер.
